    Major pieces of the web
    Browsers - a program that runs on your computer to display files found on the web
    The internet - the world's largest computer network
    HTTP - protocol to connect browser to server
    Servers - computers the host the files that make up the internet

    HTML Markup
    Consists of Tags and Content.
    Tags tell the computer to how display the contents, together this is called an Element.
    Format is <name> contents </name> or <opening tag> contents <closing tags>

    Examples of tags
    The <b> tag makes content bold
    The <em> tag makes content italicised
    The anchor tag, <a>, make links in a webpage. Ex: <a href="www.udacity.com">Udacity</a>
    where href is an attribute and www.udacity.com is the value. Udacity,
    The image tag format is as follows:
    <img src="url" alt="text">
    where url is where the image is downloaded from and the alt, or alternate, is the text that is displayed when the image fails to load.
    Image tags have no closing tag because it has no content. These are also called void tag.

    Whitespace tags
    The <br> tag creates a line break, which moves text after the tag to a new line. The <p> tag renders content in one paragraph. Since it has content, it requires a closing tag, while the <br> tag is a void tag.
    HTML code does not need to follow the action of the whitespace tags
    Inline vs Block - the br tag is an inline element, simply text.
    The <p> tag is a block element, an invisible block is created around the content of this tag
    Span and Div elements
    Simply contain their content but have the ability to edit how the content is laid out with CSS
    Span is inline while div is block. Both require closing tags.

    Strings are characters enclosed in quotes. They can be single or double as long as it is the same at the beginning and end of the characters.
    Variables hold any information desired. This can be strings, numbers, formulas, etc.
    Variables are assigned their content with the equals sign.
    As an example, we can assign variable "x" with the value of 5 as follows: x = 5.
    The "=" serves a dual role depending on the intended behaviour. As we've seen for variable assignment, but it is used to calculate a function in an arithmetic equation, such as 2 + 2 = 4.
    The plus sign also serves another role. It concatenates multiple strings together. "2"+"2" would result in "22". Or "I "+"went "+"to "+"the "+"store." would result in "I went to the store."
    Variables are useful to make code less complex and improves readability for others.

    Automate Your Page - Lesson Three

    Python functions are a set of instructions for the computer to execute. Functions take inputs, perform a procedure, then yield or return an output.
    Functions are created with the word def: followed by the code of what the function does on a new line. It is important to indent four spaces on the new line. Python uses white space to section off code into readable blocks.
    As an example, let's write a function to cube a number.
    def cube(q):
    number = q * q * q
    return number
    By itself, the function above would not provide a result. A user needs to call the function by name.
    Such as: print cube(5) would output 125.
    Once a function is written, it can be reused as often as needed in a program. This is valuable in reducing the amount of code required.

    Automate Your Page - Lesson Four

    Python's If statement checks if a condition is True or False and performs a block of code based on the resulting boolean. Like Functions and While loops, If statements are written with a colon at the end. An example If statement would be:
    a = 6
    if a > 5:
    print "Yes"
     This if statement prints a "Yes" since 6 is greater than 5. But if we had initiated a to 4, it would not print anything. That is because we didn't qualify an Else clause. The Else is written like the If, with a colon and an indented code block in the next line.
    a = 4
    if a > 5:
    print "Yes"
     else:
    print "No"
    This results in a No being printed because 4 is not greater than 5.

    The While loop is a block of code that will repeat itself until the initial condition is True. While loops are created as follows:
    zapp = 0
    while zapp < 5:
    print zapp
    zapp = zapp + 1
    The output of this function would be a list of numbers from 0 to 4. Once zapp reaches 5, 5 is not less than 5, and the while loop ends.

    Lists are a collection of items within one object. Lists are created with brackets and items are separated with commas. A list can have a list inside it, or nested. They can be assigned to variables.
    Here is an example of a list of cities: cities = ['Boston', 'Washington, DC', 'New York', 'Chicago']
    Lists are indexed like strings, so to extract an item, use its position, like so: my_city = cities[1]. This would return the list item as position 1, which is 'Washington, DC'.
    Lists are also mutable, which means they can be modified after creation. This is in contrast to strings, which are immutable.
    For my list above, replacing 'New York' with 'Miami' would be as follows: cities[2] = 'Miami'. Now cities = ['Boston', 'Washington, DC', 'Miami', 'Chicago']

    The For loop iterates over a sequence values, such as lists or strings. Think of this as For Each value in sequence, execute this block of code. The For loop is written as follows:
    for variable in sequence:
    code block
    The way it works is that each item in the sequence is assigned to the variable and the code block is executed. This repeats until all items in the sequence have been completed. Let's write a For loop that prints out the values of a list.
    for x in [1, 2, 3, 4, 5]:
    print x
    In this For loop, x is assigned to 1, then x is printed. Next, x is assigned to 2, then x is printed...etc.
    The result would be:
    1
    2
    3
    4
    5

    The Power of Abstraction - Lesson One

    Abstraction in computer science allows programmers to suppress complexity by utilizing already defined concepts. In Python, this occurs with the use of the Libraries and Modules.
    A Module is a collection of code that can be utilized but first needs to be imported into the program. This would be done via import module_name Modules only contain a single file. Libraries contain several modules within it. Libraries are imported with the following syntax from library_name import module_name The Python Standard Library is distributed with Python. This means that you don't have to import something new to use. For instance len is a function that counts the length of a string and can be called without importation.

    The Power of Abstraction - Lesson Two

    Classes are a "neatly packaged box, that puts things together really well" - Kunal. They act as a blueprint for something you're creating.
    Objects are instances or examples of the blueprint. In other words, a class Car could have multiple instances with differentiating factors, such as color or horsepower.
    Classes create an object in memory that did not previously exist and can perform actions defined within the class. Methods are functions defined within a class.
    Objection Oriented Programming, or OOP, utilizes classes and methods to create programs. This allows for large scale projects to be created since pre-written code can be leveraged and called upon at any time. OOP in python is similar to how CSS interacts with HTML. A user can create classes in CSS for how their HTML is structured and call that class when needed without rewriting the code.
    I found the following Youtube video that also explains Classes and their syntax.

    Instance variables are variables defined in the class with the self instance. Each of the same class created will have their own copies of these variables.
    Inheritance is the practice of reusing code from other already created classes in a new class. This is a key advantage in OOP. When a class inherits from another, it doesn't just inherit the instance variables but also any methods defined.
    Method Overriding is the process of creating the same method of a parent class within the inherited class. If the new method is called, it will take on its defined properties and not the properties defined in the parent class.
